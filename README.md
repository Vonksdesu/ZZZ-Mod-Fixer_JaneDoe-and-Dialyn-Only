# ZZZ-Mod-Fixer_JaneDoe-and-Dialyn-Only

This is a special tool for fixing issues with the characters Jane Doe and Dialyn.

---

## About The Tool

This is a tool to fix complex issues resulting from changes HoYoverse made specifically to the characters Jane Doe and Dialyn.

### Origin Tool and Disclaimer

This tool was created entirely by Satan1c. I simply ported it from C# to Python and added the `.exe` file for the Jane Doe character, which Satan1c did not provide on his GitHub.

Since the original tool was written in C#.

If you prefer C#, please visit his GitHub, if you want the Python version, use mine.

Full details are available on their GitHub, here, I am simply providing the Python and `.exe` versions specifically for the Jane Doe character, which they did not include.

Regarding updates, I will certainly align this tool with any updates released by Satan1c.

## How to Use

Since this is specific to the Jane Doe and Dialyn characters, the `.exe` file must be placed directly inside the respective character's mod folder rather than the main `Mods` folder, as it functions differently from the standard `zzz-mod-fixer.exe` [(ZZZ-Mod-Fixer)](https://github.com/Vonksdesu/ZZZ-Mod-Fixer/releases).

For further instructions, you can visit [Satan1c](https://github.com/Satan1c/ZZMI_tools/wiki/Jane-2.5-fix) GitHub page directly.
